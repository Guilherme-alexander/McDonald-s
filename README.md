# 🍔 McDonald's - Site Modernizado

![McDonald's Banner](https://raw.githubusercontent.com/Guilherme-alexander/McDonald-s/main/imagens/logo1.jpg)

> Um site moderno e responsivo para o McDonald's, com design contemporâneo, animações suaves e experiência de usuário otimizada.

## 📋 Sobre o Projeto

Este projeto é uma reformulação completa do site do McDonald's, trazendo uma interface moderna, intuitiva e totalmente responsiva. O site foi desenvolvido com foco em experiência do usuário, performance e design visual atraente.

### ✨ Características Principais

- 🎨 **Design Moderno**: Interface limpa com gradientes, sombras e efeitos visuais contemporâneos
- 📱 **Totalmente Responsivo**: Adapta-se perfeitamente a todos os tamanhos de tela
- 🚀 **Animações Suaves**: Scroll reveal, efeitos parallax e transições fluidas
- 📝 **Formulário Inteligente**: Validação em tempo real e feedback visual
- 📍 **Páginas Estruturadas**: Home e Contato com informações completas
- 💬 **WhatsApp Integrado**: Botão flutuante para contato direto
- 🎯 **SEO Amigável**: Meta tags otimizadas para mecanismos de busca

## 🎯 Funcionalidades Detalhadas

### 📄 Página Inicial (index.html)
- **Hero Banner**: Apresentação impactante com chamada para ação
- **Produtos em Destaque**: Grid de cards com os principais itens
- **Promoção Especial**: Banner destacado com ofertas
- **Scroll Animado**: Elementos que surgem suavemente ao rolar

### 📞 Página de Contato (contact.html)
- **Formulário Completo**: Nome, endereço, telefone, WhatsApp, pedido e observações
- **Validação Inteligente**: Verificação de campos obrigatórios
- **Feedback Visual**: Mensagens de sucesso/erro estilizadas
- **Informações da Loja**: Endereços, horários e contatos
- **Redes Sociais**: Links com ícones coloridos
- **Localizações**: Grid com todas as unidades

### 🎨 Recursos Visuais
- **Header Fixo**: Navegação sempre visível com efeito de scroll
- **Gradientes**: Cores vibrantes e modernas
- **Efeitos Hover**: Interações suaves em cards e botões
- **Botão WhatsApp**: Flutuante e sempre acessível
- **Parallax**: Efeito de profundidade no banner principal

## 📱 Responsividade

O site foi desenvolvido com abordagem **mobile-first** e se adapta a:

| Dispositivo | Breakpoint | Características |
|-------------|------------|-----------------|
| 📱 Mobile | < 480px | Menu hambúrguer, layout vertical |
| 📱 Tablet | < 768px | Ajustes de grid e espaçamento |
| 💻 Desktop | < 1024px | Layout completo com múltiplas colunas |
| 🖥️ Desktop | ≥ 1024px | Experiência completa e otimizada |

## 🛠️ Como Executar

### Opção 1: Localmente
```bash
# Clone o repositório
git clone https://github.com/Guilherme-alexander/McDonald-s.git

# Navegue até a pasta
cd McDonald-s

# Abra o arquivo index.html no seu navegador
# Ou use um servidor local (recomendado)
```

### Opção 2: Servidor Local (Recomendado)
```bash
# Com Python 3
python -m http.server 8000

# Com Node.js (Live Server)
npx live-server

# Com VS Code
# Instale a extensão "Live Server" e clique em "Go Live"
```
